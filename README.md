# ota

A simple query engine, created to explore how these kinds of systems are implemented.

## Currently implemented features

- Data types:
  - Int
  - Bool
- Operations:
  - Scan
  - Projection
- Expressions:
  - Int literal
  - Math expressions:
    - Addition
    - Subtraction
    - Multiplication
    - Division
    - Modulo
  - Boolean expressions
    - (Not) equal
    - Greater-than(-or-equal)
    - Less-than(-or-equal)
    - And
    - Or
